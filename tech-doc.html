Navigation Bar

Using a nav element I created a navigation bar at the top of the page that helps users navigate the page. Clicking a link will take you to the relevant section of the page.

This is done by giving the text a link that links to another element within the page rather than an outside source, like this:

<nav>
  <a href="[element-id]">text</a>
</nav>
  • You'll see this on most websites styled in many different ways
  • My example is for navigating one page, but it is also often used to navigate between multiple pages of a website where you would link to a separate page rather than a specific element
Unordered List

I'm using ul elements to create the bullet point sections of the page. These are called unordered lists, similarly there are also ordered lists (<ol></ol>) which are numbered rather than bullet pointed.

Within the list there are li elements (list items) and this is the text that you want to have the bullet points.

<ul>
  <li>text</li>
  <li>text</li>
</ul>
  • This text is an example of how a list item displays
Paragraph

The p element is used to write paragraphs. It is useful when you want a large body of text or a just text that is smaller than a heading or title of some kind since it is not bold or very large by default.

This section of text and others on the page like it are all p elements. Im using two separate p elements here to separate the paragraphs with spacing between them.

<p>paragraph text</p>
<p>second paragraph text</p>
  • Although it is small and undecorated by default, the paragraph text can be styled to be larger, bold, italicized, etc. like any other element
CSS

CSS is used in addition to HTML to style it. Without CSS all of the elements of the page would be plain - without color, without alignment/centering, without styled text(bold, size differences), etc.

There are many things you can do with CSS. Put simply, it is how you put elements in the place you want them, make them look the way you want them to look, and act the way you want them to act.

element {
 property: value;
}
  • The property is want you want to change about the element and the value is what you want to change the property to
  • An example would be the property "background-color" with the value then being a color that you want the elements background to change to. Heres how it would look:
body {
 background-color: red;
}
